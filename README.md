# ProblemSolution

This repository contains multiple console-based Java applications under the `com` package.

## Run the Project

Use the provided shell script from the project root:

```bash
./run.sh
```

If your shell requires it, run the script with `bash`:

```bash
bash run.sh
```

## Run A Specific Application

Pass the application name to the script to launch one module directly.

```bash
./run.sh library_management
```

## Available Applications

- `result_management`
- `library_management`
- `learntrack`
- `food_delivery`
